# FLOW株式会社 ホームページ

## 1. プロジェクト概要
電気工事・空調設備工事会社「FLOW株式会社」の公式ホームページ。
法人向け工事依頼、元請案件獲得、協力会社募集、採用応募を目的とした多目的コーポレートサイト。

**ターゲットユーザー**
- 法人企業・元請会社（工事依頼）
- 協力会社・外注パートナー
- 工場・店舗関係者
- 未経験求職者

## 2. ページ構成
- `/` - トップページ（ヒーロー・3導線・強み・CTA）
- `/services` - 事業内容
- `/about` - 会社概要
- `/recruit` - 採用情報（将来フェーズ）
- `/contact` - お問い合わせ（将来フェーズ）

- [x] トップページ（ヒーロー・導線・強み・CTA）
